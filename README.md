## About this repo

This repository captures my day-by-day learning progress in Python. It contains the code I write, the notes I take, and the practice questions I solve. The goal is to build competence through consistent practice and to create a searchable record I (and others) can revisit.

**Goal:** Becoming proficient in Python for data analysis and practical problem solving.

## Repo structure

- `codes/` — runnable scripts and daily code snippets (examples: exercises, small projects).
- `notes/` — short write-ups, explanations, and insights from learning sessions.
- `practice-questions/` — solved problems and challenges (with inputs/outputs and brief solutions).
- `README.md` — overview + progress tracker.

If your actual folders differ, adapt the above to match your structure (for example, `Basic/`, `List/`, `String/` etc.).


- To add a new entry: append a row with `Day`, `Date`, `Topic`, `Status` and a relative path to code or notes.
- Use `✅` for done, `⏳` for in-progress, and `📝` for planned.
